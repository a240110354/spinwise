# Paraphraser MVP 需求文档

**版本**：V1.0  
**日期**：2026-03-31  
**状态**：初稿

---

## 1. 产品概述

### 产品名称
Paraphraser（暂定）

### 产品类型
AI 驱动的文本改写/重述工具 Web 应用

### 核心价值
帮助用户快速将文本改写为不同风格，无需注册即可使用

### 目标用户
- 写作者、文案人员需要改写润色
- 学生党论文降重/改写
- 跨境电商卖家多语言表达
- 任何需要改写文本的人

---

## 2. 功能需求

### 2.1 核心功能（Must Have）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 文本输入 | 支持粘贴或输入文本，字数统计显示 | P0 |
| 改写执行 | 调用 AI API 完成改写 | P0 |
| 结果展示 | 展示改写后的文本 | P0 |
| 一键复制 | 复制改写结果到剪贴板 | P0 |
| 改写风格选择 | 提供 3-5 种风格选项（见下方） | P0 |
| 加载状态 | 改写过程中显示 loading | P0 |

**改写风格选项：**
1. **正式** — 商务、书面语
2. **口语化** — 轻松、日常
3. **简洁** — 更简短表达
4. **扩展** — 更详细丰富
5. **学术** — 论文风格

### 2.2 增强功能（Should Have）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 原文/结果对比 | 左右分栏对比 | P1 |
| 字数变化统计 | 显示原文/结果字数差 | P1 |
| 错误提示 | API 失败、超长文本等提示 | P1 |

### 2.3 未来功能（Nice to Have）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 改写程度控制 | 滑块控制改写幅度（保守 ↔ 激进） | P2 |
| 多语言支持 | 输入/输出语言选择 | P2 |
| 历史记录 | 本地存储最近改写 | P2 |
| 批量改写 | 多段文本批量处理 | P3 |

---

## 3. 非功能需求

### 3.1 性能
- 首屏加载 < 2秒
- 改写响应时间 < 10秒（取决于 API）

### 3.2 兼容性
- 响应式设计，适配桌面/平板/手机
- 浏览器：Chrome、Safari、Firefox、Edge 最新两个版本

### 3.3 安全
- 不存储用户输入文本（纯 API 调用）
- 无需注册，无敏感数据

---

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端框架 | Next.js | 支持 SSR/SSG，部署方便 |
| 样式 | Tailwind CSS | 快速开发，体积小 |
| AI API | Claude API / OpenAI API | 按 token 计费 |
| 部署平台 | Cloudflare Pages | 免费、全球 CDN、自动 HTTPS |
| 域名 | 自有域名（可选） | - |

### 4.2 架构设计

```
用户浏览器
    │
    ▼
Cloudflare Pages (静态托管)
    │
    ▼
AI API (Claude/OpenAI) ── 直接从浏览器调用
    │
    ▼
返回改写结果
```

> **注意**：纯前端架构，不需要后端服务器。AI API 调用直接从浏览器发起，需配置 API Key 保护（可使用 Cloudflare Workers 做代理）。

### 4.3 数据流

1. 用户输入文本 → 前端
2. 前端 → AI API（携带 API Key）
3. AI API → 前端（改写结果）
4. 前端展示结果

### 5.2 交互流程

2. 输入或粘贴文本

5. 展示改写结果
6. 点击「复制」复制结果

## 6. API 设计（前端调用）

### 6.1 改写接口

**请求**
```javascript
POST /api/paraphrase
Content-Type: application/json

{
  "text": "需要改写的文本",
  "style": "formal" | "casual" | "concise" | "expand" | "academic"
}
```

**响应**
```javascript
{
  "success": true,
  "result": "改写后的文本",
  "stats": {
    "originalLength": 100,
    "resultLength": 120
  }
}
```

**错误响应**
```javascript
{
  "success": false,
  "error": "文本过长，请控制在 5000 字以内"
}
```

## 7. 边缘情况处理

| 场景 | 处理方式 |
|------|----------|
| 空文本提交 | 按钮禁用，或提示「请输入文本」 |
| 文本过长 | 提示限制，建议分段处理 |
| API 调用失败 | 显示友好错误提示「改写失败，请重试」 |
| 网络断开 | 提示「网络连接异常」 |
| 并发多次点击 | 防抖处理，loading 期间禁用按钮 |
